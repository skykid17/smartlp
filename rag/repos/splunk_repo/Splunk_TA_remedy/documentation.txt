The Splunk Add-on for BMC Remedy allows a Splunk software administrator to use BMC Remedy REST APIs to collect incident, audit, and incident work log data. This add-on for BMC Remedy integrates Remedy with your Splunk platform and enables a Splunk software administrator to create and update incidents in a Remedy system from the Splunk platform, either through search commands or the alert action provided by the add-on.
