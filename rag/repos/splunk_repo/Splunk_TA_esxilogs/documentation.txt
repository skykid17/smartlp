The Splunk Add-on for VMware ESXi Logs contains the input stanzas to receive the data from the syslog and search-time/index-time extractions and parses and extracts the fields from the VMware ESXi logs. It accepts ESXi log data using syslog which allows you to troubleshoot events and host issues. Download the Splunk Add-on for VMware ESXi Logs from Splunkbase at https://splunkbase.splunk.com/app/5603/.
