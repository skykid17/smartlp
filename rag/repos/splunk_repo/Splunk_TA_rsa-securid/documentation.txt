The Splunk Add-on for RSA Authentication Manager allows a Splunk software administrator to collect data from the RSA Authentication Manager (AM) server via syslog. This add-on provides the inputs and CIM-compatible knowledge to use with other Splunk apps, such as Splunk Enterprise Security . See the RSA documentation for more information. 

Download the Splunk Add-on for RSA SecurID from Splunkbase at http://splunkbase.splunk.com/app/2958.
